# Solana Devnet Faucet

Simple Solana devnet faucet utility for testing and Web3 onboarding.

## Features

- Fast faucet request
- Beginner friendly
- Lightweight utility
- Web3 onboarding support
- Simple wallet input

## Purpose

This project is designed to simplify access to Solana devnet tokens for beginners and developers.

## Stack

- Node.js
- Express
- Solana Web3.js

## Status

Work in progress.
